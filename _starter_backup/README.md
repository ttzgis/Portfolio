# GIS Portfolio Starter

Einfache statische Portfolio-Seite fuer GitHub Pages.

## Dateien

- `index.html`: Inhalt und Struktur der Seite
- `styles.css`: Layout und Design
- `script.js`: Kleine Animationen und Jahreszahl im Footer

## Fuer GitHub Pages

Wenn diese Dateien im Hauptverzeichnis eines GitHub-Repositories liegen, kann GitHub Pages sie direkt als Website veroeffentlichen.

## Naechste sinnvolle Anpassungen

1. Namen, E-Mail und GitHub-Link in `index.html` ersetzen
2. Projektkarten durch echte GIS-Projekte austauschen
3. Optional Screenshots oder Webkarten ergaenzen
